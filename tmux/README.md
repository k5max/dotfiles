# tmux 配置

## 目录结构

.tmux.conf

## 插件列表

无

## 快捷键

| command              | keybinding      |
| -------------------- | --------------- |
| leader键             | \<C-a>          |
| 重新加载配置文件     | \<leader>r      |
| pane水平分割         | \<leader>\|     |
| pane垂直分割         | \<leader>-      |
| pane最大化           | \<leader>z      |
| 切换到左边pane       | \<leader>h      |
| 切换到右边pane       | \<leader>l      |
| 切换到下边pane       | \<leader>j      |
| 切换到上边pane       | \<leader>k      |
| 切换到最后使用的pane | \<leader>e      |
| 与前一个pane交换位置 | \<leader>\<C-u> |
| 与后一个pane交换位置 | \<leader>\<C-d> |
| 向左调整pane边缘5行  | \<leader>H      |
| 向右调整pane边缘5行  | \<leader>L      |
| 向下调整pane边缘5行  | \<leader>J      |
| 向上调整pane边缘5行  | \<leader>K      |
